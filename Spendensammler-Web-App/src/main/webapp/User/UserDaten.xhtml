Datenauswertung-Benutzer

In dieser Ansicht finden Sie mehrere Diagramme mittels derer Sie die Spendendaten Auswerten können.

Spendensumme pro Monat

Dieses Diagramm gibt die Spendensumme für jeden Monat eines Jahres aus, Sie können das auszuwertende Jahr selber angeben und mittels des Button 'Update' die neuen Werte anzeigen lassen.
Beachten Sie dabei, dass die Jahresangabe nicht größer als das aktuelle Jahr sein darf und nicht kleiner als 2015.






Spendensumme pro Jahr

Dieses Diagramm gibt die Spendensumme für mehrere Jahre aus, Sie können die auszuwertenden Jahre selber angeben und mittels des Button 'Update' die neuen Werte anzeigen lassen.
Beachten Sie dabei, dass die Jahresangaben nicht größer als das aktuelle Jahr sein dürfen und nicht kleiner als 2015. Außerdem darf das Jahr 'Von' nicht größer als das Jahr 'Bis' sein.
